A powerful plugin for applying filters to various WordPress elements with custom field support and dynamic tag replacement.

== Description ==

FilterFlex allows you to modify how WordPress displays various elements like titles, content, excerpts, and more. Create sophisticated filters with custom field integration, taxonomy support, and dynamic tag replacement.

Key Features:

* Filter various WordPress elements (titles, content, excerpts, etc.)
* Support for custom fields with dynamic value replacement
* Taxonomy integration including categories and custom taxonomies
* Location rules for precise filter application
* Live preview of filter results
* Priority-based filter ordering
* Transformation options (case changes, search/replace, etc.)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/filterflex` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the FilterFlex menu item to configure your filters

== Frequently Asked Questions ==

= Can I filter custom post types? =

Yes, FilterFlex works with any post type registered in WordPress.

= Does it support custom taxonomies? =

Yes, you can enable any registered taxonomy in the plugin settings.

= Can I use custom fields in my filters? =

Yes, FilterFlex includes full support for custom fields with an easy-to-use interface.
